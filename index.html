📚 Arrays Overview

▼

🔹 What is an Array?

  • An array is a collection of elements stored in contiguous memory
  • Example: int[] numbers = {10, 20, 30, 40, 50};

✅ Advantages

  • Stores multiple values in one variable
  • Fast index access
  • Enhances readability and efficiency

⚠️ Limitations

  • Fixed in size once declared
  • Elements must be of the same data type

🛠️ Declaring & Initializing Arrays

int[] arr = new int[5]; // Size-based initialization
int[] arr = {10, 20, 30};   // Direct initialization

🔁 Common Array Operations

📏 Get Array Length

int length = array.length;

🔁 Looping Through Arrays

  • For Loop:
for (int i = 0; i < array.length; i++) {
  // Process array[i]
}
  • Enhanced For Loop:
for (int item : array) {
  // Process item
}

🔍 Searching Example

for (int num : arr) {
  if (num == key) {
    found = true;
    break;
  }
}

🔧 Arrays Utility Class

import java.util.Arrays;
Arrays.sort(arr);
Arrays.binarySearch(arr, key);
Arrays.toString(arr);

🔢 2D Arrays (Multidimensional)

int[][] matrix = {
  {1, 2, 3},
  {4, 5, 6}
};

// Access:
int value = matrix[1][2]; // value is 6

// Traversing:
for (int i = 0; i < matrix.length; i++) {
  for (int j = 0; j < matrix[i].length; j++) {
    System.out.print(matrix[i][j] + " ");
  }
}

🔍 Searching & Sorting Algorithms

▼

Searching Algorithms

1. Linear Search

  • Concept: Sequentially checks each element
  • Time Complexity: Best: O(1), Worst/Average: O(n)
for (int i = 0; i < arr.length; i++) {
  if (arr[i] == key) return i;
}

2. Binary Search

  • Concept: Divide and conquer (for sorted arrays only)
  • Time Complexity: Best: O(1), Worst/Average: O(log n)
while (low <= high) {
  int mid = (low + high) / 2;
  if (arr[mid] == key)
    return mid;
  else if (arr[mid] < key)
    low = mid + 1;
  else
    high = mid - 1;
}

⏱ Big O Notation (Time Complexity)

Complexity Description Example
O(1) Constant time Accessing an element
O(log n) Logarithmic time Binary search
O(n) Linear time Linear search
O(n²) Quadratic time Bubble sort
O(n log n) Efficient for large datasets Quick sort

Sorting Algorithms

1. Bubble Sort

  • Compares and swaps adjacent elements
  • Complexity: O(n²)
for (i = 0; i < n - 1; i++)
  for (j = 0; j < n - i - 1; j++)
    if (arr[j] > arr[j+1])
      swap(arr[j], arr[j+1]);

2. Insertion Sort

  • Builds a sorted array one element at a time
  • Complexity: O(n²)
for (i = 1; i < n; i++) {
  key = arr[i];
  j = i - 1;
  while (j >= 0 && arr[j] > key)
    arr[j+1] = arr[j--];
  arr[j+1] = key;
}

3. Selection Sort

  • Repeatedly finds the smallest element and moves it to the sorted portion
  • Complexity: O(n²)
for (i = 0; i < n - 1; i++) {
  min = i;
  for (j = i + 1; j < n; j++)
    if (arr[j] < arr[min])
      min = j;
  swap(arr[min], arr[i]);
}

🧠 Key Concepts

  • Use Linear Search for unsorted arrays
  • Use Binary Search for sorted arrays
  • Understand the trade-offs in efficiency and simplicity for sorting algorithms

6. Conditions & If Statements

Conditional statements make your program behave differently based on data.

age = 17

if age < 13:
    print("Child")
elif age < 18:
    print("Teen")
else:
    print("Adult")

Structure: if handles the first condition, elif specifies additional cases, and else catches all other situations.

7. Loops

Loops let you repeat code.

For Loop

Used when the number of iterations is known.

for i in range(3):
    print(i)
While Loop

Used when repeating until a condition becomes false.

count = 0
while count < 3:
    print(count)
    count += 1

8. break and continue

break

Exits the loop early.

for i in range(5):
    if i == 3:
        break
    print(i)  # prints 0,1,2
continue

Skips the rest of the current iteration and continues with the next.

for i in range(5):
    if i == 3:
        continue
    print(i)  # prints 0,1,2,4
