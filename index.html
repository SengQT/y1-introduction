🔹 What is an Array?

  • An array is a collection of elements stored in contiguous memory
  • Example: int[] numbers = {10, 20, 30, 40, 50};

✅ Advantages

  • Stores multiple values in one variable
  • Fast index access
  • Enhances readability and efficiency

⚠️ Limitations

  • Fixed in size once declared
  • Elements must be of the same data type

🛠️ Declaring & Initializing Arrays

int[] arr = new int[5]; // Size-based initialization
int[] arr = {10, 20, 30};   // Direct initialization

🔁 Common Array Operations

📏 Get Array Length

int length = array.length;

🔁 Looping Through Arrays

  • For Loop:
for (int i = 0; i < array.length; i++) {
  // Process array[i]
}
  • Enhanced For Loop:
for (int item : array) {
  // Process item
}

🔍 Searching Example

for (int num : arr) {
  if (num == key) {
    found = true;
    break;
  }
}

🔧 Arrays Utility Class

#include <algorithm>
std::sort(arr, arr + n);
int pos = std::binary_search(arr, arr + n, key);

🔢 2D Arrays (Multidimensional)

int matrix[2][3] = {
  {1, 2, 3},
  {4, 5, 6}
};

// Access:

// Traversing:

  for (int j = 0; j < 3; j++) {
    std::cout << matrix[i][j] << " ";
  }

}

🔍 Searching & Sorting Algorithms

▼

Searching Algorithms

1. Linear Search

  • Concept: Sequentially checks each element
  • Time Complexity: Best: O(1), Worst/Average: O(n)
for (int i = 0; i < arr.length; i++) {
  if (arr[i] == key) return i;
}

2. Binary Search

  • Concept: Divide and conquer (for sorted arrays only)
  • Time Complexity: Best: O(1), Worst/Average: O(log n)
while (low <= high) {
  int mid = (low + high) / 2;
  if (arr[mid] == key)
    return mid;
  else if (arr[mid] < key)
    low = mid + 1;
  else
    high = mid - 1;
}

⏱ Big O Notation (Time Complexity)

Complexity Description Example
O(1) Constant time Accessing an element
O(log n) Logarithmic time Binary search
O(n) Linear time Linear search
O(n²) Quadratic time Bubble sort
O(n log n) Efficient for large datasets Quick sort

Sorting Algorithms

1. Bubble Sort

  • Compares and swaps adjacent elements
  • Complexity: O(n²)
for (i = 0; i < n - 1; i++)
  for (j = 0; j < n - i - 1; j++)
    if (arr[j] > arr[j+1])
      swap(arr[j], arr[j+1]);

2. Insertion Sort

  • Builds a sorted array one element at a time
  • Complexity: O(n²)
for (i = 1; i < n; i++) {
  key = arr[i];
  j = i - 1;
  while (j >= 0 && arr[j] > key)
    arr[j+1] = arr[j--];
  arr[j+1] = key;
}

3. Selection Sort

  • Repeatedly finds the smallest element and moves it to the sorted portion
  • Complexity: O(n²)
for (i = 0; i < n - 1; i++) {
  min = i;
  for (j = i + 1; j < n; j++)
    if (arr[j] < arr[min])
      min = j;
  swap(arr[min], arr[i]);
}

🧠 Key Concepts

  • Use Linear Search for unsorted arrays
  • Use Binary Search for sorted arrays
  • Understand the trade-offs in efficiency and simplicity for sorting algorithms

Additional Beginner Concepts

1. Compiling From Command-Line

What It Means

Compiling from the command-line involves using a compiler directly through a terminal rather than an IDE. This offers more control over compiler options and the build process.

How It Works

  • Compiler Command: 
    g++ -std=c++17 -o hello hello.cpp

    Each part means:

    • g++: The C++ compiler command.
    • -std=c++17: Use C++17 standard.
    • -o hello: Name the output executable “hello.”
    • hello.cpp: The source file to compile.
  • Running the Executable: 
    ./hello

    This command executes the compiled program.

Why It’s Useful

  • Customizability: Add multiple flags and options.
  • Automation: Easily integrate into build scripts and CI systems.
  • Portability: Consistent across various operating systems.

2. Basic Language Constructs

Overview

These constructs are the core building blocks of any C++ program. They include variables, operators, and control structures that govern data processing and program flow.

Key Components

  • Variables and Data Types: Store information using primitive types (int, float, etc.).
  • Operators: Arithmetic, relational, and logical operators enable computation and decisions.
  • Control Structures: if statements and loops (for, while, do-while) help automate repeated tasks and decision making.

Applications: Create programs that perform calculations, check conditions (e.g., even/odd), and iterate over collections.

3. Structures, Unions, and Enumerations

Structures: Group related variables into one data type for clearer code.

struct Point {
    int x;
    int y;
};

int main() {
    Point p = {10, 20};
    std::cout << "x: " << p.x << ", y: " << p.y << std::endl;
    return 0;
}

Unions: Allow different types to share the same memory for efficiency.

union Data {
    int i;
    float f;
    char str[20];
};

int main() {
    Data data;
    data.i = 10;  
    std::cout << "data.i: " << data.i << std::endl;
    return 0;
}

Enumerations: Define a type with a set of named constants to improve code clarity.

enum Color { RED, GREEN, BLUE };
Color myColor = RED;

Combined Benefits: Structure complex data, optimize memory use, clarify program intent, and build robust applications.

2. Intermediate Level

a. Functions, Pointers, and References

Functions: Support overloading, default arguments, and recursion.

int add(int a, int b) { return a + b; }
double add(double a, double b) { return a + b; }

Pointers: Allow direct memory access.

int value = 5;
int *ptr = &value;
std::cout << "Value: " << *ptr;

Dynamic Memory: Allocate and deallocate memory using new and delete.

int *arr = new int[10];
// Use the array...
delete[] arr;

References: Alias to existing variables for efficient passing.

int original = 100;
int &ref = original;
ref = 200;  // original becomes 200

b. Object-Oriented Programming (OOP)

Classes vs. Structures: Classes default to private members and support encapsulation.

class Rectangle {
private:
    int width, height;
public:
    Rectangle(int w, int h) : width(w), height(h) {}
    int area() const { return width * height; }
};

Inheritance and Polymorphism: Enable code reuse and dynamic behavior.

class Animal {
public:
    virtual void speak() const { std::cout << "Animal sound\n"; }
};

class Dog : public Animal {
public:
    void speak() const override { std::cout << "Woof!\n"; }
};

Animal *animal = new Dog();
animal->speak(); // Woof!
delete animal;

c. The Standard Template Library (STL)

Containers: Use vectors, lists, maps, and sets for efficient data storage.

#include <vector>
std::vector<int> numbers = {1, 2, 3, 4};
numbers.push_back(5);

Iterators and Algorithms: Traverse containers and perform common operations.

for (auto it = numbers.begin(); it != numbers.end(); ++it) {
    std::cout << *it << " ";
}

Lambda Expressions: Write inline anonymous functions.

auto square = [](int x) { return x * x; };
std::cout << square(5);  // Outputs: 25

d. Memory Management and Error Handling

Manual Memory Management: Use new/delete while avoiding memory leaks.

Smart Pointers: Manage dynamic memory safely.

#include <memory>
std::unique_ptr<int> smartInt = std::make_unique<int>(42);

Exception Handling: Use try-catch blocks to handle errors.

try {
    throw std::runtime_error("An error occurred");
} catch (const std::exception &e) {

3. Advanced Level (Expert Topics)

a. Modern C++ Features (C++11/14/17/20)

Advanced Templates: Explore function and class templates, specialization, and meta-programming.

Lambda Expressions (Advanced):

int val = 5;
auto func = [val]() mutable { val++; return val; };
std::cout << func();  // Outputs: 6

Move Semantics and Perfect Forwarding:

class Buffer {
public:
    Buffer(Buffer&& other) noexcept { /* move resource */ }
    Buffer& operator=(Buffer&& other) noexcept { /* move resource */ return *this; }
};

Concurrency and Parallelism: Use the thread library and synchronization primitives.

#include <thread>
#include <mutex>
std::mutex mtx;
void printBlock(int n, char c) {
    std::lock_guard<std::mutex> lock(mtx);
    for (int i = 0; i < n; ++i)
        std::cout << c;
    std::cout << std::endl;
}
